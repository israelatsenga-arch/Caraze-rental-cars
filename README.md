# Caraze Rental Cars

## Project Description 
This website enables users to rent luxurious cars instead of buying them.

## Author
Mark Israel

## Built with
* HTML
* CSS
* JAvaScript

## Project Setup Instructions
1. Clone the repository.
2. Open your index.html in your web browser.
3. Go through the inventory and read through the car details to be able to select your vehicle of choice.
4. Enter your name,Id number,email,the number of days you want to rent the car and select the vehicle you want.
5. Click 'submit' then get your confirmation message.

## BDD(Behaviour Driven Development)
1. Input:name,identification card number,email,the number of days you want to rent the car and the vehicle you want.
    * Output:A message informing you on how much you are supposed to pay and requesting you to visit one of the branches to make payments and get your car
